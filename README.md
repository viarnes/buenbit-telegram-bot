# Buenbot
Telegram bot: @buen2bot

1. Install: `npm install`
2. Set: `TELEGRAM_BOT_TOKEN` & `ETHGASSTATION_API_KEY`
2. Run: `node index.js`

# Bot commands

## Get pairs
Fetch price tickers from Buenbit.

- Usage: `/par {ticker}`
- Example: `/par daiars`

## Blog search
Search term on Buenbit Blog.

- Usage: `/blog {term}`
- Example: `/blog defi`

## Gas price
Fetch gas price (in gwei) from ETH Gas Station

- Usage: `/gas`
